Hypervisor based computer virtualization:

You're probably like; "Woah, that's a mouthload. Slow the fuck down crazy dog."
And you're well within your right to be so, let me break it down for you:

So, you can host a piece of server software on a single machine, right? Well, what I'm doing is hosting alot of different server software,
on a single machine, via some neat software a company called VMware created ESXi. Software like ESXi is called a "Hypervisor".
I, via a subdomain on this domain, host my own ESXi homelab to do simple tasks that I need. Like file servers, or a DNS packet interception Server.

Here's a list of what I currently run on my ESXi server:

  • NextCloud (File Server)

    • General purpose file server for backing up my artwork, documents and etc. Opensource, and fully customizable.

  • OpenVPN End Node

    • An endnode that allows access to my local home network, permitting me to work on my homelab from anywhere in the world.

  • Pihole via Debian

    • This was the DNS Packet Interception server I was talking about. Basically drops DNS query packets to ad servers and effectively blocks ads on my local network.

  • Subsonic Server (Music)

    • My foul attempt at hosting my music online for easy-access. Basically decided to just use Spotify because it's easier.

  • Various Game Servers

    • I turn these on from time to time. Depends on what I'm doing. Nice little way to have an on-demand game server for whatever me and my friends want to play.
    And with unlimited subdomains, I just make a new subdomain every time we pick a new game.

That's basically my little homelab, although I really want to get the ball rolling on another appliance to host some networking softwared called "PFSense". But I lack the required NIC Hardware.
TBH, just need to get my paws on a compatible Intel NIC and should be golden.

Feel free to poke around or whatever, but I keep everything updated 24/7 on a cron job. Best of luck. Oh, and if you do find something, let me know and I might have a little treat for you :3
